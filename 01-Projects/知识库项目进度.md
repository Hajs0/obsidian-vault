## 📝 使用方法

### 启动服务器

```bash
cd /home/ubuntu/knowledge-base-project
python3 web/serve.py 8080
```

### 访问页面

- 知识库：http://10.0.0.5:8080
- 知识图谱：http://10.0.0.5:8080/graph.html

### 创建知识关联

在 Markdown 文件中使用双向链接：

```markdown
---
related: ["相关页面1", "相关页面2"]
---

# 当前页面

相关内容请参考 [[相关页面1]] 和 [[相关页面2]]。
```

### 迁移知识库

直接复制 `vault/` 目录即可：
```bash
cp -r vault/ /目标路径/
```
