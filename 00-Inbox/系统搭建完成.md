## 🚀 快速开始

### 1. 安装 Obsidian

**macOS**:
```bash
brew install obsidian
```

**Windows**:
```powershell
winget install Obsidian.Obsidian
```

**Linux**:
```bash
wget https://github.com/obsidianmd/obsidian-releases/releases/download/v1.5.0/Obsidian-1.5.0.AppImage
chmod +x Obsidian-1.5.0.AppImage
./Obsidian-1.5.0.AppImage
```

### 2. 打开 Vault

1. 打开 Obsidian
2. 点击 "Open folder as vault"
3. 选择 `~/obsidian-vault` 目录

### 3. 安装插件

1. Settings → Community plugins
2. 关闭 "Safe mode"
3. 点击 "Browse" 搜索并安装：
   - **Dataview** - 数据查询
   - **Calendar** - 日历视图
   - **Kanban** - 看板视图
   - **Templater** - 高级模板

---

## 📝 模板使用

### 日记模板

1. 点击左侧日历图标
2. 或使用快捷键：`Ctrl/Cmd + P` → "Daily notes: Open today's daily note"

### 笔记模板

1. 创建新笔记
2. `Ctrl/Cmd + P` → "Templates: Insert template"
3. 选择模板：
   - `note-template` - 普通笔记
   - `project-template` - 项目笔记
   - `resource-template` - 资源笔记
   - `meeting-template` - 会议记录

## ☁️ 同步与备份

### Git 同步

```bash
# 手动同步
cd ~/obsidian-vault
./Scripts/sync.sh

# 自动同步（每小时）
crontab -e
# 添加：0 * * * * ~/obsidian-vault/Scripts/sync.sh
```

### 备份

```bash
# 手动备份
cd ~/obsidian-vault
./Scripts/backup.sh

# 自动备份（每天）
crontab -e
# 添加：0 2 * * * ~/obsidian-vault/Scripts/backup.sh
```
