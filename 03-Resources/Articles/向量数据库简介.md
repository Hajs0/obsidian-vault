# 向量数据库简介

## 什么是向量数据库？

向量数据库是专门用于存储和检索高维向量的数据库系统，广泛应用于 AI 语义搜索和 [[RAG 检索增强生成|RAG 系统]]。

## 常用向量数据库

### Chroma
- **类型**：开源嵌入数据库
- **特点**：简单易用，四函数 API
- **适用**：小型项目、快速原型
- **GitHub**：⭐ 16k+

### FAISS
- **类型**：开源相似性搜索库
- **特点**：Facebook 开发，支持 GPU 加速
- **适用**：大规模向量检索
- **GitHub**：⭐ 32k+

### Pinecone
- **类型**：托管向量数据库
- **特点**：完全托管，自动扩展
- **适用**：生产环境、企业级应用

### Qdrant
- **类型**：开源向量搜索引擎
- **特点**：Rust 开发，高性能
- **适用**：需要高性能的场景
- **GitHub**：⭐ 22k+

## 在 RAG 中的应用

向量数据库是 [[RAG 检索增强生成|RAG 系统]] 的核心组件：

1. **存储阶段**：将文档向量存入数据库
2. **检索阶段**：根据查询向量检索相似文档
3. **生成阶段**：将检索结果作为上下文
