# 🌐 浏览器操作技能

## 概述

Hermes Agent 内置浏览器操作能力，支持自动化网页操作。

## 核心工具

| 工具 | 功能 |
|------|------|
| `browser_navigate` | 导航到 URL |
| `browser_snapshot` | 获取页面快照 |
| `browser_click` | 点击元素 |
| `browser_type` | 输入文本 |
| `browser_scroll` | 滚动页面 |
| `browser_vision` | 视觉分析 |
| `browser_console` | 执行 JavaScript |

## 常用模式

### 1. 表单填写

```python
# 导航到页面
browser_navigate(url)

# 获取快照找到元素
browser_snapshot()

# 填写表单
browser_type(ref="@e1", text="用户名")
browser_type(ref="@e2", text="密码")

# 提交
browser_click(ref="@e3")  # 点击提交按钮
```

### 2. 数据抓取

```python
# 导航
browser_navigate(url)

# 使用 JavaScript 提取数据
data = browser_console(expression="document.querySelectorAll('.item').length")
```

### 3. 视觉验证

```python
# 截图并分析
browser_vision(question="页面是否加载完成？")
```

## 最佳实践

1. **每次操作后检查状态**：使用 `browser_snapshot()` 确认
2. **处理错误**：使用 `browser_back()` 回退
3. **等待加载**：检查 console 有无错误
4. **使用 ref ID**：元素标识如 `@e1`, `@e2`

## 常见问题

**Q: 元素找不到？**
A: 滚动页面后重新 `browser_snapshot()`

**Q: 页面加载慢？**
A: 检查网络，或使用 `browser_console` 检查错误
