# 📖 知识库使用指南

## 快速开始

### 1. Web 访问

访问地址: http://10.0.0.5:8080

功能:
- 📚 浏览所有笔记
- 🔍 全文搜索
- 📝 查看笔记详情

### 2. Obsidian 编辑

1. 安装 [Obsidian](https://obsidian.md/)
2. 打开 vault 目录
3. 开始编辑

### 3. 添加新笔记

在 `vault/knowledge/` 下创建 `.md` 文件:

```markdown
---
title: 笔记标题
created: 2026-05-27
tags: [tag1, tag2]
---

# 笔记标题

内容...
```

## 目录结构

```
vault/
├── Home.md              # 首页
├── daily/               # 每日笔记
├── knowledge/           # 知识分类
│   ├── ai/              # AI 相关
│   ├── hermes/          # Hermes 技能
│   ├── coding/          # 编程
│   └── business/        # 业务
└── templates/           # 模板
```

## 迁移方法

### 方法 1: 直接复制

```bash
cp -r vault/ /目标路径/
```

### 方法 2: Git

```bash
cd /home/ubuntu/knowledge-base-project
git init
git add .
git commit -m "备份知识库"
git push
```

## 与 Hermes Agent 集成

在对话中可以直接:
- 询问知识库内容
- 让 Agent 添加笔记
- 同步 Hermes 技能

## 技巧

1. **使用标签**: `tags: [ai, rag]` 便于搜索
2. **双向链接**: `[[笔记名]]` 链接相关笔记
3. **每日笔记**: 在 `daily/` 记录学习
4. **定期备份**: 使用 Git 版本控制
