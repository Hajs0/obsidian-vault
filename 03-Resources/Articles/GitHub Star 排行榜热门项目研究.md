## 🔥 关键趋势分析

### 1. AI 本地化趋势明显
- **Ollama** (120K+) 成为 2024-2025 年增长最快的项目
- 用户强烈需求本地、隐私的 AI 解决方案

- **Tabby**、**n8n** 等自托管工具增长迅速

### 5. 知识管理 + AI 结合
- **Khoj**、**STORM** 等项目将 AI 与个人知识管理结合

## 🚀 快速开始指南

### 1. 安装 Ollama

```bash
# macOS/Linux
curl -fsSL https://ollama.com/install.sh | sh

# Windows
# 下载安装程序: https://ollama.com/download
```

### 2. 运行本地模型

```bash
# 运行 Llama 2
ollama run llama2

# 运行 Mistral
ollama run mistral

# 运行 CodeLlama
ollama run codellama
```

### 3. 安装 Open WebUI

```bash
# Docker 部署
docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main
```

### 4. 安装 Khoj

```bash
# pip 安装
pip install khoj

# 或 Docker 部署
docker run -d -p 42110:42110 khoj/khoj
```

### 5. 安装 Flowise

```bash
# npm 安装
npm install -g flowise

# 或 Docker 部署
docker run -d -p 3000:3000 flowiseai/flowise
```
